# Table style workflow

Tables are handled as a separate design layer.

Presets:

- academic_three_line: formal paper tables.
- journal_compact_table: dense appendix tables.
- dataframe_zebra: report tables.
- dashboard_table: dashboard tables.
- editorial_table: editorial explanatory tables.
- minimal_table: minimal white tables.
- dark_table: slide tables with dark background.
- pastel_table: soft explanatory tables.
- monochrome_print_table: print-safe tables.

Rules:

- Use sparse rules for paper tables.
- Use zebra rows for reports.
- Use compact settings for appendices.
- Keep numeric precision and alignment consistent.
