# Skills

This directory contains portable Agent Skills packages.

Current skill:

- `scientific-figure-making/`: publication-ready scientific figure generation with automatic palette selection.

Claude Code project-local copy:

- `.claude/skills/scientific-figure-making/`

The `.claude/` copy makes the skill directly invocable inside this repository as `/scientific-figure-making`. The top-level `skills/` copy is for agents and platforms that consume a generic open skills directory or for copying into another project.
